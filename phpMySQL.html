HEAD FIRST PHP & MYSQL

Preço: R$ 100,00

Sinopse

Se você está pronto para criar páginas mais complexas do que aquelas que você pode construir com HTML e CSS, o Head First PHP e MySQL é o guia de aprendizado final para a construção de sites dinâmicos baseados em banco de dados usando PHP e MySQL. Embalado com exemplos do mundo real, este livro ensina todos os elementos essenciais da programação do lado do servidor, desde os fundamentos do PHP e da codificação MySQL até tópicos avançados, como validação de formulário, IDs de sessão, cookies, consultas de banco de dados e associações, E / S de arquivo Operações, gerenciamento de conteúdo e mais. O primeiro PHP e MySQL oferece o mesmo formato visualmente rico que transformou cada título na série Head First em um best-seller, com muitos exercícios, questionários, quebra-cabeças e outros recursos interativos para ajudá-lo a reter o que Você aprendeu. Use o PHP para transformar páginas HTML estáticas em sites dinâmicos. Recupere e preencha suas próprias tabelas de banco de dados MySQL e trabalhe com dados armazenados em arquivos. Força consultas sofisticadas do MySQL com associações e aperfeiçoe seus resultados com LIMIT e ORDER BYUse cookies e sessões para Rastrear informações de login dos visitantes e personalizar o site para usuáriosProteger seus dados de ataques de injeção SQLUse expressões regulares para validar informações em formuláriosDynamicamente exibir texto b Sobre as informações da sessão e criar imagens no flyPull dados sindicados de outros sites usando PHP e XML. Ao longo do livro, você criará exemplos sofisticados - incluindo uma lista de correspondência, uma placa de trabalho e um site de namoro online - para ajudá-lo a aprender Como aproveitar o poder do PHP e do MySQL em uma variedade de contextos. Se você está pronto para construir um site verdadeiramente dinâmico, Head First PHP e MySQL é a maneira ideal de começar.
